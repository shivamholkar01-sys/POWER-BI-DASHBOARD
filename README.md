# Student Performance Factors Analysis

This project analyzes the **Student Performance Factors** dataset to identify the key factors influencing students' exam scores. The dataset contains **6,607 student records** with **20 features**, including study hours, attendance, parental involvement, access to resources, sleep hours, previous scores, motivation level, tutoring sessions, family income, teacher quality, school type, and exam scores.

## Objectives

* Analyze the relationship between student habits and academic performance.
* Identify the most influential factors affecting exam scores.
* Perform data cleaning and exploratory data analysis (EDA).
* Create insightful visualizations and dashboards.
* Generate actionable recommendations using data analytics.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* SQL (optional)
* Power BI (for dashboard visualization)

## Dataset Features

* Study Hours
* Attendance
* Previous Scores
* Sleep Hours
* Motivation Level
* Tutoring Sessions
* Family Income
* Teacher Quality
* School Type
* Exam Score (Target Variable)

This project demonstrates practical data analysis, visualization, and business analytics skills using a real-world educational dataset.
